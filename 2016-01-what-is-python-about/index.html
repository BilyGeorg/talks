Who uses it?

Google, NASA, Red Hat, Dropbox, Mozilla, Pinterest, Venmo, etc.

Data scientists, web developers, sys admins, tinkerers, learners, etc.

How can I use it?

  • Make a website (with Django, Pyramid, Flask)
  • Analyze your data (with Pandas, NumPy, SciPy, Matplotlib)
  • Automate tasks (with the standard library, Ansible, Fabric)

What it is not.

  • For systems that have real time requirements
  • For embedded systems (although, watch out for MircoPython!)
